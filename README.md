# Dragon ball Z 

This example has an GraphQL API of characters of Dragon Ball Z.
[See the proyect](https://crisronda.github.io/dbz-angular/dbz/) 

# Libraries

- Angular framework
- Apollo for Graphql subscriptions
- Angular Material 
- [Backend project](https://github.com/CrisRonda/graphql-express)

Configuration Graphql in Real Time with subscriptions when add and delete
